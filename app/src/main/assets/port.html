The program accepts the following parameters:

Parameter Required Description Example
-th no Number of Threads to run -th 32
-t no Timeout for Socket disconnection -t 100

Warning: Reccommended to use the Threads according to the speed of your internet connection and not the computational capacity of your device. Some tests show that with 60Mbps in download and 15Mbps in upload it is possible to use 32 Threads without having any problems. A connection with 500Mbps in download and 200Mbps in upload can also handle 800 Threads.
